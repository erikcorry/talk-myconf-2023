# Erik Corry

## A career in language runtimes

* 1998: BETA: A pioneering OO language from Scandinavia
* 2004: OOVM: Smalltalk for embedded systems
* 2007: V8: The JavaScript engine that powers Chrome and Node.js
  - Irregexp: JITted regular expressions in JavaScript
* 2010: Dart: A new language for the web (and mobile)
  - Dart2js: A Dart to JavaScript compiler
  - Dart VM: A Dart virtual machine
  - Dartino: A Dart VM for microcontrollers
* 2018: Toit - a high level language for IoT
